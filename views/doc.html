Les requettes POST

L'url ci-dessous permet d'enregistrer un nouvel utilisateurs dans notre API

https://brianboudrioux.fr/simplon/api/users

Pour cela vous devez passer en parametres de votre requettes POST les valeurs correspondant au champs suivant :

username, email, password

Exemple (POSTMAN) : 
L'url ci-dessous permet de vérifier la présence d'un utilisateur dans notre API, 
 très utile pour effectuer une connexion !

https://brianboudrioux.fr/simplon/api/connect

Pour cela vous devez passer en parametres de votre requettes POST les valeurs correspondant au champs suivant :

email, password

Exemple (POSTMAN) : 
L'url ci-dessous retourne tout les films/series appartenant aux catégories spécifier en parametres

https://brianboudrioux.fr/simplon/api/products/categories

Pour cela vous devez passer en parametres de votre requettes POST les valeurs correspondant au champs suivant :

categories[]

Exemple (POSTMAN) : 
L'url ci-dessous permet d'enregistrer une nouvelle category dans notre API

https://brianboudrioux.fr/simplon/api/categories

Pour cela vous devez passer en parametres de votre requettes POST les valeurs correspondant au champs suivant :

name, description, picture

Exemple (POSTMAN) : 
L'url ci-dessous permet d'enregistrer un nouveau film/serie dans notre API 
 (!important! ne pas oublier de spécifier la ou les categories liée au film)

https://brianboudrioux.fr/simplon/api/products

Pour cela vous devez passer en parametres de votre requettes POST les valeurs correspondant au champs suivant :

name, description, picture, media, categories[]
